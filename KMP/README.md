# KMP

KMP算法是一个很经典的串匹配算法，我们首先提出问题：

给定一个文本串T，一个模式串P，判定T中是否存在某一子串与P相同，若存在，报告子串在T中的起始位置。

本章节的代码中给出了蛮力算法和KMP算法的实现，可以通过预处理命令来使用不同的算法。

**理解KMP算法的关键是理解next表！**

next表是一个串索引的数组，除了第一个元素之外，其他每个元素代表的含义是：若匹配过程中在模式串的索引i处发生失配，则i应该回退到next[i]处；

上面是在操作层面上理解next表，那么next表的本质是什么呢？

**next表的本质是：若next[i] = t，则在模式串P[0, i)中，自匹配的真前缀和真后缀的最大长度为t**

KMP算法中使模式串可以快速跃进的关键就是这个长度为t的自匹配真前缀和真后缀！
